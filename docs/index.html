Blitz ⚡

Self-hosted speech-to-text for your Mac. Hold a hotkey, talk, get text pasted into the focused app. Bring your own OpenAI key.

Download installer Drag the file into Claude Code → done in 2 minutes.

How to install

  1. Click Download installer above. You'll get blitz-setup.md.
  2. Open Claude Code on your Mac. (Setup guide below if you don't have it yet.)
  3. Drag blitz-setup.md into Claude Code's input and press Enter.

Claude Code takes over: clones the source, asks for your OpenAI API key, builds the app with Xcode, installs it to /Applications, walks you through permissions.

Don't have Claude Code yet?

2-minute setup. Open Terminal:

brew install node
npm install -g @anthropic-ai/claude-code
cd ~/Desktop && claude

Once Claude Code is open, drag blitz-setup.md into the input.
